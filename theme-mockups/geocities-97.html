Virtualization from scratch

July 3, 2023

There is something deeply satisfying about building a virtual machine from the ground up. Not downloading one, not configuring a cloud instance — actually constructing the abstraction layer that sits between software and silicon. You start with nothing but a bare set of CPU instructions and the conviction that you can trick a program into believing it has an entire computer to itself. It is part systems programming, part illusion, and part stubbornness. Every register you virtualize, every interrupt you intercept, pulls you closer to understanding what a computer really is under the layers of abstraction we take for granted every day.

In this post I walk through the process of building a minimal hypervisor, handling memory translation, I/O trapping, and the strange edge cases that make you question everything you thought you knew about how operating systems work.
